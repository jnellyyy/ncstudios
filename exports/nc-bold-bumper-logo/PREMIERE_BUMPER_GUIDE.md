# NC Studio Bracket Bumper for Premiere Pro

Main direction: right-aligned `[nc]  NC STUDIO` lockup, with a custom stretched block `nc` mark. No PHOTO + FILM line.

## Best static assets
- `nc-bold-bracket-right-lockup-black-card.png`
- `nc-bold-bracket-right-lockup-cream-card.png`
- `nc-bold-bracket-right-lockup-transparent.png`
- `nc-bold-bracket-right-lockup-white-transparent.png`
- `nc-bold-bracket-right-lockup.svg`

## Premiere sequences
Folders are in `premiere_sequences/`.

1. File > Import
2. Open a sequence folder
3. Select only `frame_0001.png`
4. Tick `Image Sequence`
5. Import
6. Interpret as `15 fps`

Use `00_nc_studio_right_lockup_black_flash` as the main bumper.
Use `00b_nc_studio_right_lockup_overlay_flash` above footage.

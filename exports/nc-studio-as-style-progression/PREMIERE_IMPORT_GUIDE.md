# NC Studio AS-Style Progression Bumper

This version follows the reference progression: full `[ nc studio ]`, then the middle drops away into `[ n   c ]`, then it tightens into `[nc]`.

## Premiere Pro

Import one of these as an Image Sequence by selecting `frame_0001.png` and ticking Image Sequence:

- `premiere_sequences/16x9_black_card` for normal landscape edits.
- `premiere_sequences/16x9_transparent_overlay` to place over footage.
- `premiere_sequences/9x16_black_card` for reels/TikTok vertical edits.
- `premiere_sequences/9x16_transparent_overlay` for vertical overlays.

Interpret footage at 24fps. The sequence is just over 2 seconds, so you can trim the start or end in Premiere.

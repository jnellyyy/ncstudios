# NC Studios Premiere Pro Transition Pack

These are transparent PNG image sequences for Premiere Pro.

## Import

1. In Premiere Pro, go to File > Import.
2. Open one transition folder.
3. Select only `frame_0001.png`.
4. Tick `Image Sequence`.
5. Import.
6. Right click the imported sequence, choose Modify > Interpret Footage.
7. Set frame rate to `15 fps`.
8. Put the sequence on a video track above your footage.

## Use

- For the transition overlays, place the clip over the edit point between two shots.
- Because the files are transparent PNGs, `Normal` blend mode works.
- For extra punch, try Opacity > Blend Mode > `Screen` or `Linear Dodge (Add)`.
- Trim, speed up, reverse, or duplicate the sequence in Premiere to fit the cut.

## Included

- `01_gold_flare_sweep_transition`: diagonal champagne flare wipe
- `02_gold_flicker_flash_transition`: soft camera-flash flicker
- `03_champagne_edge_light_leak_transition`: edge light leak sweep
- `04_gold_shutter_flicker_transition`: subtle shutter-line flicker
- `05_nc_logo_wow_intro`: cinematic NC logo intro with gold particles and flare

All sequences are `1920 x 1080` with transparency.

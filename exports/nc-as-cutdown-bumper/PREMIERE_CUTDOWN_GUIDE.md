# NC Studio Cut-Down Bumper

This is the Adult-Swim-style version: full NC Studio identity first, then a hard cut down into the compact [nc] mark.

## Main Premiere Imports

1. Import `premiere_sequences/01_full_to_nc_black_card/frame_0001.png` as an Image Sequence for a black title card.
2. Import `premiere_sequences/02_full_to_nc_transparent_overlay/frame_0001.png` as an Image Sequence when you want it over footage.
3. Interpret both sequences at 24fps.
4. Trim the end blink if you want it to finish on the mark instead of blinking out.

# NC Studio Type Guide

## Font Files

| Use | File | Notes |
| --- | --- | --- |
| Brand display | `assets/fonts/NC_Font_.ttf` | Heavy custom NC font. Use for the `[nc]` mark, short logo-style words, and tiny brand accents only. |
| Lighter brand accent | `assets/fonts/NC_Font_Lighter.ttf` | Slightly lighter saved version. Use for short labels like `[nc studio]`, eyebrow text, and small navigation accents. |
| Normal website text | System text font | Use for paragraphs, forms, descriptions, client-facing copy, and anything longer than a few words. |

## Website Roles

| Role | Use For | Font |
| --- | --- | --- |
| Header | Main hero title and large section titles | `--font-heading` |
| Subheader | Supporting intro lines, card titles, short section explanations | `--font-heading` or `--font-body` depending on length |
| Normal text | Paragraphs, forms, notes, service descriptions, client copy | `--font-body` |
| Brand accent | Navigation labels, eyebrow labels, `[nc studio]` opener | `--font-brand-light` |
| Logo mark | Actual logo files, not typed live text where possible | PNG logo assets |

## Rules

- Do not use the heavy NC font for paragraphs.
- Do not use the heavy NC font for long hero sentences.
- Use the lighter NC font only for short brand accents.
- Keep normal website copy readable first.
- The website opener should read `[nc studio]` first, then cut down to `[nc]`.
- The app can use the compact `[nc]` mark in small top bars.
